# ChessConsoleGame

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-CLI-orange.svg?logo=terminal)](https://en.wikipedia.org/wiki/Command-line_interface)

Консольная шахматная игра на Kotlin с Unicode-отображением фигур и пошаговым управлением.

## О проекте

Шахматная игра для командной строки с полной визуализацией доски и фигур с использованием Unicode символов. Игра поддерживает пошаговые ходы двух игроков.

**Основной функционал:**
- Полноценная шахматная доска 8x8
- Unicode-символы для всех фигур (белые и черные)
- Поочередные ходы игроков
- Валидация допустимых ходов
- Интуитивный ввод ходов (например: "e2 e4")
- Визуальное отображение текущего хода

## Технологии

- Kotlin
- Объектно-ориентированное программирование
- Unicode символы для визуализации
- Модульная архитектура с разделением на фигуры
- Валидация игровых правил
